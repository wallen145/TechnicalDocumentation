Paragraph



The p-tag tag class of this display theme causes text to be indented at the start of the paragraph.

Here is an example of the p element with no styling:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo magnam mollitia, quam placeat deserunt quos nostrum, asperiores esse animi qui reprehenderit quis repellendus ratione dolorum? Modi dolorum inventore suscipit voluptate.

Here is an example using p element with the p-tag styling:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Officiis quia totam voluptatem, dignissimos laudantium deserunt tempora! Facilis veniam at sint similique iure veritatis quae voluptas, quod placeat doloremque animi libero.

Anchor



The a-tag class for the anchor tag displays the content with a specific color scheme depending on the the state of othe link. When the link is unvisited, it is colored blue, after it is visited it is green, but when the cursor hovers over the link it will be hot red, and when the link is active it will be brown.
